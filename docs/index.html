JSON und JavaScript

Wenn Sie auf den Button drücken, dann wird das unten abgebildete JSON-Dokument als JavaScript-Objekt erzeugt. Es wird dann ein Dialog angezeigt, der einige Werte des Objekts beinhaltet. 

{
	"titel"       : "Großes Buch der Allwissenheit",
	"autoren"     : [ "Alice Mayer", "Bob Brown" ],
	"jahr"        : 2001,
	"erstauflage" : true,
	"verlag"  : {
		"name" : "Beispiel-Verlag",
		"ort"  : "Berlin"
	}
}
